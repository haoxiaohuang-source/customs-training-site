📌 什麼是 D/O？

👉 D/O = Delivery Order

👉 白話就是：領貨要用的文件，小提單 / 提貨單。

🎯 用途

  • 作為提領貨物的重要文件
  • 是換單後取得的文件
  • 沒有 D/O，通常無法辦理後續提貨

🪜 D/O 怎麼來？

  1. 先有 B/L（提單）
  2. 向船公司 / 代理行辦理換單
  3. 繳相關費用
  4. 取得 D/O

👉 白話：拿提單去換可領貨的文件。

🔍 新人要注意什麼

  • 換單地點
  • 換單期限
  • 要帶什麼文件（正本提單 / 電放文件等）
  • 費用是否已確認

⚠️ 常見錯誤

  • 不知道去哪裡換單
  • 提單條件不符合，無法順利換單
  • 忽略時效，超過期限
  • 費用沒先確認清楚

🧩 B/L 跟 D/O 的關係

B/L：提單，是運送與交貨的文件。

D/O：換單後取得，用來辦理提貨。

👉 新人記法：先看 B/L，再換 D/O。

🧠 新人記住一句話

👉 D/O = 「領貨用的文件」
